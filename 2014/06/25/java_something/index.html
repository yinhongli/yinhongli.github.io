java基础

关于类

1.定义类名称时，开头得首字母大写，java规范.

2.java中声明一个类得方式有两种，public class 类名称和class类名称。使用public class类名称声明一个类时，类名称必须与文件名称一致，否则程序无法编译。使用class类名称声明一个类时，类名称可以与文件名称不一致，但是执行时一定要执行生成后的class。

3.一个java文件中可以有多个class，但只能有一个public class。

关于字节

4.java8种基本数据类型：数值型－－整数类型（byte，short，long，int），数值型－－浮点类型（double，float）
字符型（char），布尔型（boolean）

5.引用数据类型： 类，接口，数组

6.long 64位，也就是8个字节（bytes），int 32位，也就是4个字节。short两个字节。

7.数据类型的溢出，当最大值加上1时，结果变成表示范围中最小的值。

8.integer

9.字符类型，计算机处理字符类型时，是把这些字符当成不同的整数来看待，严格来说，字符类型也算整数类型的一种。

10.unicode为每个字符制定了一个唯一的数值。在任何的语言，平台，程序中都可以安心使用，java使用的就是unicode字符码系统。

10.浮点数 32字节，double 64字节。

11.自动类型转换。字节少的与字节多的运算，向字节多的转换。

12.String，任何数据类型碰到String类型的变量或常量之后都向String类型转换。

13.数据类型的强制转换。
