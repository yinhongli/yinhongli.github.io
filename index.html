内部类：类中定义一个类。可以声明为public或者private。

使用static可以声明内部类，用static声明的内部类变成了外部类，但是用static声明的内部类不能访问非static的外部类属性。

面向对象－－－高级

继承： class 子类 extends 父类｛｝，只允许多层继承，而不能多重继承。继承的子类有时称为派生类。子类不能直接访问父类的私有成员，子类可以调用父类中的非私有方法，但是不能直接调用父类中的私有成员。

子类对象的init过程，子类对象在init之前必须先调用父类的构造方法后再调用子类自己的构造方法。

方法的覆写：子类定义了与父类同名的方法，但是在方法覆写时必须考虑到权限，被子类覆写的方法不能拥有比父类方法更加严格的访问权限。

不过子类为什么要定义跟父类名称一样的方法呢？

private<default<public 如果父类中方法时public 子类必须也是public，否则程序无法编译。 子类权限只能放大。

private方法不能覆写。

如果想要继续调用父类中被覆写过的方法，使用super关键字就可以。

final关键字

  • 使用final声明的类不能有子类

  • 使用final声明的方法不能被子类所覆写

  • 使用final声明的变量既成为常量，常量不可以修改

抽象类

java中可以创建一种类专门用来当作父类，这种类为“抽象类”，抽象类的作用类型模板，其目的是要设计者依据它的格式来修改并创建新的类，但是并不能直接由抽象类创建对象，只能通过抽象类派生出新的类，再由它来创建对象。

  • 包含一个抽象方法的类必须是抽象类

  • 抽象类和抽象方法都要使用abstract关键字声明

  • 抽象方法只需声明而不需要实现

  • 抽象类必须被子继承，子类（如果不是抽象类）必须覆写抽象类中的全部抽象方法

  • 抽象方法不要使用private声明

接口

java基础

关于类

1.定义类名称时，开头得首字母大写，java规范.

2.java中声明一个类得方式有两种，public class 类名称和class类名称。使用public class类名称声明一个类时，类名称必须与文件名称一致，否则程序无法编译。使用class类名称声明一个类时，类名称可以与文件名称不一致，但是执行时一定要执行生成后的class。

3.一个java文件中可以有多个class，但只能有一个public class。

关于字节

4.java8种基本数据类型：数值型－－整数类型（byte，short，long，int），数值型－－浮点类型（double，float）
字符型（char），布尔型（boolean）

5.引用数据类型： 类，接口，数组

6.long 64位，也就是8个字节（bytes），int 32位，也就是4个字节。short两个字节。

7.数据类型的溢出，当最大值加上1时，结果变成表示范围中最小的值。

8.integer

9.字符类型，计算机处理字符类型时，是把这些字符当成不同的整数来看待，严格来说，字符类型也算整数类型的一种。

10.unicode为每个字符制定了一个唯一的数值。在任何的语言，平台，程序中都可以安心使用，java使用的就是unicode字符码系统。

10.浮点数 32字节，double 64字节。

11.自动类型转换。字节少的与字节多的运算，向字节多的转换。

12.String，任何数据类型碰到String类型的变量或常量之后都向String类型转换。

13.数据类型的强制转换。

关于堆栈

  • 栈（操作系统）：由编译器自动分配释放，存放函数的参数值，局部变量的值等。其操作方式类似于数据结构中的栈。栈使用的是一级缓存，他们通常都是被调用时处于存储空间中，调用完后立即释放

  • 堆（操作系统）：一般由程序员分配释放，若程序员不释放，程序结束后可能由os回收，分配方式类似于链表。堆存放于二级缓存中，生命周期由虚拟机的垃圾回收算法来决定，所以调用这样对象的速度相对来低一些

  • 堆栈就是java对内存的划分，是个逻辑概念，jvm实现的逻辑划分，程序运行的时候操作这两个区域完成代码的运行。
    栈的个数多，空间小，一个栈默认都是几百K才，每个线程都有自己的栈，里面存的是局部变量；
    堆，整个jvm就只有一个，多个线程共享这一个堆，空间很大上G，所有new出来的对象都放在这里，

关于方法

  • 方法的命名，首字母小写，之后的每个单词首字母大写。

  • 方法无返回值，使用void关键字声明

  • static关键字

  • 方法的重载：方法的名称相同，但参数的类型和参数的个数不同，通过传递参数的个数及类型的不同可以完成不同功能的方法调用。

  • 方法的递归调用：一种特殊的调用形式，是方法自己调用自己。尽量避免使用递归调用。

java新性

  • 可变参数

  • foreach输出

面向对象

  • 封装性：两次含义，把对象的属性和行为看成一个密不可分的整体，将这两者封装在一个不可分割的独立单位（对象）中。另一层是指信息隐蔽，把不需要让外界知道的信息隐藏起来。

  • 继承性：被继承的类称为父类或超类，而继承产生的类称为子类或派生类。

  • 多态性：方法重载，对象多态（子类对象可以与父类对象进行相互转换，而且根据其使用的子类的不同，完成的功能也不同）

关于类

类中定义的方法，没有加上static关键字，是因为此时定义的方法将由对象调用，而不象之前那样与主方法定义在一个类重且由主方法直接调用。

类名称，属性定义，方法定义。

所有的对象名称都在栈内存中保存，而对象的具体内容则保存在对应的堆内存之中。

  • 对象使用之前必须实例化，否则会报错，空指针异常。

  • 对于私有属性要使用setter和getter方法设置和访问实际上还有一个原因，就是在java中存在反射机制。

构造方法

  • 构造方法的名称必须与类名称一致

  • 构造方法的声明处不能有任何返回值类型的声明

  • 不能在构造方法中使用return返回一个值
